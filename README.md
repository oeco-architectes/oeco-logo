Oeco Architectes logo
=====================

Requirements
------------

- [Node.js]
- [Sketchtool]

Getting started
---------------

### Install

```bash
npm install
```

### Build

```bash
npm run build
```

This exports all slices from [src/oeco-logo.sketch] into [dist/] directory.
